IndexedDB


es una interfaz de programación de aplicaciones (API) de JavaScript proporcionada por navegadores web para administrar una base de datos NoSQL de objetos JSON. Es un estándar mantenido por el World Wide Web Consortium (W3C). El soporte para IndexedDB se agregó a Firefox versión 4,4​ Google Chrome versión 11,5​ e Internet Explorer versión 10.6​ Safari agregó soporte en la versión 8.7​ Web SQL Database fue una API previa desarrollada por Apple. Pero Firefox se negó a agregarle soporte y argumentó en contra de que se convirtiera en un estándar porque codificaría las peculiaridades de SQLite. Por lo tanto, quedó en desuso a favor de IndexedDB.

Caracteristicas de IndexedDB


  • Almacenamiento de pares clave-valor.
  • Es asincrónico.
  • Asuntos de apoyo.
  • Restricción de homología.
  • Gran espacio de almacenamiento
  • Soporte de almacenamiento binario.

Uso de IndexedDB


IndexedDB es una manera de almacenar datos dentro del navegador del usuario. Debido a que permite la creación de aplicaciones con habilidades de consulta enriquecidas, con independencia de la disponibilidad de la red, sus aplicaciones pueden trabajar tanto en línea como fuera de línea.
